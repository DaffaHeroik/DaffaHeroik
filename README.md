## 👨‍💻 Tentang Saya

Saya **Daffa Heroik**, seorang **developer** dan **inovator teknologi tepat guna** asli NTB, Indonesia. Fokus saya pada pengembangan **payment gateway**, **REST API**, serta proyek **IoT berbasis ESP32**. Saya juga seorang **Linux enthusiast** yang gemar mengelola server, VPS, dan otomatisasi jaringan.

- 🔭 Saat ini mengerjakan: **Heroik Payment** & **Malawali Payment**
- 🌱 Terus belajar: **Cloud Architecture**, **Cybersecurity**, **Home Assistant**
- 🤝 Tertarik kolaborasi: proyek open-source yang berdampak sosial dan edukasi digital
- ⚡ Fun fact: Saya bisa menyusun script bash sambil mendengarkan The 1975 🎧
